Building exhibits

Overview

Several online exhibit-building tools incorporate spatial data. In this module, we’ll review some of the most common examples used in digital humanities projects (circa 2017).

Neatline

This Omeka add-on was designed to build place-based narratives that are well-integrated with Omeka collections. It is maintained by the Scholars’ Lab at the University of Virginia Library.

  • Example: Georges E. Michaud : Souvenirs d’un grand homme by Lysanne Brazeau, Sophie Cayer and Ariane Pelletier as part of the ISI6354 Cultural Heritage Resources course taught by Jada Watson.
  • Example: Shakespearean street names and places in Stratford, Ontario by Nagin Zabihi, Celine Baker and Ruth Truong as part of the ENG 4142 seminar course “How Shakespeare Became ‘Shakespeare’” taught by Irene Makaryk and Amanda Montague.

Features

  • Can use web maps, scanned images and georectified maps for the background. For more information about including maps you’ve georectified using Map Warper, see this tutorial by Kristen Mapes.
  • Integrates well with items that have been uploaded and described in Omeka.
  • Can annotate the background image with points, lines, and polygons.
  • Has an optional timeline feature.

Installation

  • Requires Omeka, which in turn requires a server environment with PHP (5.2.11 or greater), MySQL (5.0 or greater), and Apache.
  • The University of Ottawa Library hosts Omeka sites for researchers who are affiliated with the University. Please contact me for more information.

StoryMap JS

This platform for building map-based exhibits was developed by the Knight Lab at Northwestern University. It works in your web browser and can be accessed with a Google account.

  • Example: A Company of Fools: The tradition of Shakespeare in the park and its reception in Ottawa, Ontario, Canada by Katelyn Boros, Alexandria Jacobs and Stephanie Kramer as part of the [ENG 3133 “Elizabethan Shakespeare” course]](http://biblio.uottawa.ca/omeka1/shakespearecanada/) taught by Irene Makaryk and Amanda Montague.
  • Example: Shakespeare in the North by Maggie Kendall, Andrea Zehr and Krystyna Dodds as part of the ENG 4142 seminar course “How Shakespeare Became ‘Shakespeare’” taught by Irene Makaryk and Amanda Montague.

Features

  • Options for image-based backgrounds and web maps.
  • Easy to use, no installation required.

Installation

  • Hosted by the Knight Lab. Exhibits can be embedded on websites.
  • Requires a Google account to log in.

Additional tools

  • Palladio is a suite of data visualization tools that allow you to map the relationships between different objects.
  • Esri Story Maps allows you to create compelling narratives using real geospatial data. It is integrated with the ArcGIS Online platform and features several templates.
